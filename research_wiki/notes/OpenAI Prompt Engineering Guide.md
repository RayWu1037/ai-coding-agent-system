# OpenAI Prompt Engineering Guide

## Summary

The OpenAI prompt engineering guide explains that better results come from clear instructions, explicit output expectations, and iterative refinement. It recommends placing instructions early in the prompt, being specific about the desired format, and treating prompting as a process of controlled improvement rather than a one-shot request. This source is useful for our project because it supports the claim that prompt quality behaves like specification quality.
