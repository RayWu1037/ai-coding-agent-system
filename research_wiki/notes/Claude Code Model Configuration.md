# Claude Code Model Configuration

## Summary

The Claude Code model configuration documentation explains how model aliases such as sonnet, opus, and haiku work, and how those settings can be controlled through startup flags, environment variables, and project settings. It also explains that model choice affects planning and execution behavior, not just naming. This source is useful for our project because early instability came from outdated or implicit model defaults.
