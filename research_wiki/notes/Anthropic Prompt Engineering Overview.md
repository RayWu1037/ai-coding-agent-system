# Anthropic Prompt Engineering Overview

## Summary

Anthropic's prompt engineering overview says prompt optimization should begin with clear success criteria and some way to test results empirically. It also recommends direct instructions, examples when needed, and structured prompting for more complex tasks. This source is useful for our project because it matches the way we improved the coding harness: we had to define success more clearly, validate generated artifacts, and stop relying on vague quality judgments.
